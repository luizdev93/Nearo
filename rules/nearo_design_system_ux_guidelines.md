# Nearo — Design System & UX Guidelines

## 1. Design Philosophy
Nearo must feel:
- Modern
- Friendly
- Fast
- Trustworthy
- Local-first

The experience should communicate simplicity and accessibility while maintaining a premium marketplace feel.

Design Principles:
- Clarity over decoration
- Large touch targets
- Fast visual scanning
- Minimal friction
- Consistent hierarchy

---

## 2. Brand Foundation

### Primary Visual Direction
Inspired by the Nearo logo:
- Purple → Blue gradient
- Soft rounded shapes
- Clean whitespace
- Friendly marketplace tone

Avoid harsh contrasts or aggressive colors.

---

## 3. Color System

### Primary Gradient (Brand Core)
Use for main actions and highlights.

Gradient:
- Start: #7A5CFF (Purple)
- Mid: #5A6BFF
- End: #47C2FF (Baby Blue)

Usage:
- Primary buttons
- Active states
- Selected tabs
- Call-to-action areas

### Primary Solid
#6A64FF

### Secondary Accent
#47C2FF

### Background
- App Background: #F7F8FC
- Card Background: #FFFFFF

### Text
- Primary: #1A1D29
- Secondary: #6B7280
- Placeholder: #9CA3AF

### Status Colors
- Success: #22C55E
- Warning: #F59E0B
- Error: #EF4444

---

## 4. Typography

Font Style:
- Clean geometric sans-serif

Recommended:
- Inter (default)

Hierarchy:
- H1: 28px / Bold
- H2: 22px / SemiBold
- H3: 18px / SemiBold
- Body: 16px / Regular
- Caption: 13px / Regular

Line height: 1.4–1.6

---

## 5. Spacing System

Base unit: 4px

Spacing scale:
- 4
- 8
- 12
- 16
- 20
- 24
- 32
- 40

Screen padding: 16px

---

## 6. Corner Radius

Use soft rounded UI:
- Buttons: 14px
- Cards: 18px
- Inputs: 12px
- Modal sheets: 24px (top only)

---

## 7. Elevation & Shadows

Soft shadows only.

Card Shadow:
- 0 6px 18px rgba(0,0,0,0.06)

Floating Elements:
- 0 10px 28px rgba(0,0,0,0.10)

---

## 8. Buttons

### Primary Button
- Gradient background
- White text
- Height: 52px
- Radius: 14px

States:
- Default
- Pressed (opacity 0.9)
- Disabled (#D1D5DB)

### Secondary Button
- White background
- Border: 1px solid #E5E7EB

### Icon Button
- Circular
- Soft background (#F1F3FF)

---

## 9. Cards

ListingCard:
- Image top
- Rounded 18px
- Title bold
- Price highlighted
- Location + time muted

Card padding: 12–16px

---

## 10. Inputs

Style:
- White background
- Subtle border
- Focus: gradient outline

Height: 48px

---

## 11. Navigation

Bottom Tab Bar:
- Floating style
- Rounded container
- Active icon uses gradient

Top Header:
- Minimal
- Large titles

---

## 12. Chat UI

My messages:
- Gradient bubble
- White text

Other messages:
- Light gray bubble

Radius: 20px

---

## 13. Icons

Style:
- Rounded
- Minimal line icons
- 24px default

Active icons may use gradient fill.

---

## 14. Motion & Animation

Animation duration:
- Fast: 120ms
- Normal: 200ms
- Modal: 280ms

Use:
- Fade
- Slide up
- Scale subtle

Avoid complex animations.

---

## 15. Image Handling

- Use rounded images
- Lazy load
- Skeleton placeholders

---

## 16. Accessibility

- Minimum touch size: 44px
- Contrast ratio AA
- Large readable typography

---

## 17. Dark Mode (Optional Later)

Not required for MVP.

---

## 18. Visual Identity Rules

Always:
- Use gradient for primary actions
- Maintain soft corners
- Keep UI airy and uncluttered
- Prefer cards over lists

Never:
- Use full-screen intrusive elements
- Use hard black backgrounds
- Overload screens with text

---

End of Design System
